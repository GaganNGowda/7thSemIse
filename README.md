> Command to execute Openmp programs:

```
$ gcc -fopenmp <file>.c  
$ ./a.out  
```
> Command to execute MPI programs:

```
$ mpicc <file>.c
$ mpirun -np 4 ./a.out
```

> Command to install R package:

```R
$ > install.packages("<package-name>")
```

> Command to R programs:

```
$ R -f <file>.R
```
